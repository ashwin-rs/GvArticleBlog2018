== Description ==

Display Page or Post Details to show page or post details like (title, content, feature image, category name, author name, custom fields, tags) using a post or page id on another page or post whatever you want. 

Major features in Display Page or Post Details include:

*   It can check all the details of page or post details, which has been included on this plugin like content if the content part
	was not there then it does not show anything on that part.
*   Just put the shortcode with page or post id like this (**[dpopdDisplay id=your-page-or-post-id]**) on the other page.

*   Here you can add your custom css if required.
